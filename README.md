## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- Yarn
  ```sh
  npm install --global pnpm@6.32.11
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/fyfirman/money-manager-mobile.git
   ```
2. Install NPM packages with Yarn
   ```sh
   pnpm i
   ```
